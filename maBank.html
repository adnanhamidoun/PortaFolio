Bienvenido a M&A Bank

Crear M&A Bank fue uno de los mayores retos que he afrontado. Era mi primer proyecto de esta escala y supuso salir totalmente de mi zona de confort. Quise combinar lo aprendido en proyectos pequeños en algo más ambicioso y real.

Tuve que unir distintos mundos: backend, frontend, bases de datos y comunicación entre servicios. Cada parte debía interactuar de forma coherente, segura y eficiente. Aprendí más enfrentando errores que leyendo tutoriales.

Hoy M&A Bank representa más que una app bancaria: es la prueba de que, con dedicación, se puede construir desde cero algo funcional, profesional y escalable. Fue el salto esencial para seguir creciendo.

Acerca de M&A Bank

M&A Bank es una plataforma bancaria web segura, moderna y funcional, diseñada para simular una experiencia real de banca digital. Combina múltiples tecnologías para ofrecer un sistema completo, intuitivo y eficiente.

Funciones destacadas

  • 🔐 Login con autenticación segura
  • 🖥️ Interfaz de usuario clara, moderna y responsive
  • 💸 Ingreso y retiro de dinero en tiempo real
  • 🌍 Cuenta multidivisa con saldos en distintas monedas
  • 🔁 Envío de dinero entre usuarios registrados
  • 💱 Conversión de divisas con tasa actualizada
  • 📊 Visualización de movimientos y transacciones
  • 💳 Generación de tarjeta de crédito virtual
  • 📈 Seguimiento de criptomonedas con gráficas en tiempo real
  • 🤖 Chatbot con IA para asistencia y soporte

Características Técnicas del Sistema

  • ✅ Priorización de programación funcional para obtener máxima eficiencia, con abstracción mediante interfaces y separación en dominios (cuentas, usuarios, transacciones…)
  • 🔒 Seguridad robusta: uso de DTOs, Spring Security, validación con tokens JWT y cifrado de datos sensibles en la base de datos
  • 🌐 Integración con servicios externos: conexión con APIs de criptomonedas en tiempo real y conversor de divisas para actualizar saldos dinámicamente
  • 🤖 Asistencia inteligente: integración de un chatbot en Python con FastAPI que aporta respuestas y soporte automatizado
  • 🎨 Frontend de vanguardia: desarrollo en Angular con componentes standalone para una interfaz limpia, modular e intuitiva
  • ⚡ Rendimiento optimizado: consultas mejoradas con SQL que garantizan un alto rendimiento y tiempos de respuesta mínimos
